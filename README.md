# 微信公众号 HTML 插入器

> 配合 **[云中书 YunType](https://yuntype.pages.dev)** 排版工具使用 —— 在微信公众号编辑器中一键插入精美 HTML 排版，告别平淡无奇的公众号样式。

---

## ✨ 功能特性（v4 侧边源码版）

- **常驻侧边面板**：编辑器右侧只留一个 `</> HTML` 小标记，点开是完整的源码编辑面板，收起即看文章效果（类似壹伴）
- **双向源码编辑**：打开面板自动**读取文章当前真实 HTML**；改完「应用到文章」写回；再打开看到的就是调整后的实时代码 —— 告别"插入完就改不动"
- **页面内快捷工具**：直接注入到公众号编辑器工具栏，提供云中书、读取、应用 HTML、图片库、一键排版等按钮
- **原生编辑区实时同步**：监听公众号 ProseMirror 编辑区变化，手动改内容/格式、原生粘贴图片后，源码面板和图片库会跟随最新 DOM
- **实时预览**：编辑框下方即时渲染效果，应用前先确认
- **🖼 图片库（解决 HTML 内插图难题）**：扫描文章里已上传的图片（合法的微信 CDN 链接），点缩略图即把 `<img src="…">` 插到光标处，想放哪放哪、随意排版，发布不会丢图
- **模板片段**：内置常用卡片/引用/按钮等，可保存自定义模板
- **可拖拽改宽 + 快捷键**：面板宽度可拖动调整，`Alt+H` 一键开关
- **代码安全过滤**：自动剥离 `<script>` / `<iframe>` / 事件处理器等危险标签

## 🚀 快速上手

### 第一步：安装插件

**Chrome 浏览器**

1. 下载本仓库（点右上角 `Code → Download ZIP`，解压）
2. 地址栏输入 `chrome://extensions/` 回车
3. 右上角开启 **开发者模式**
4. 点击 **加载已解压的扩展程序**，选择解压后的文件夹
5. 安装完成 ✅

**Edge 浏览器**

1. 同上解压，地址栏输入 `edge://extensions/` 回车
2. 左下角开启 **开发人员模式**
3. 点击 **加载解压缩的扩展**，选择文件夹
4. 安装完成 ✅

---

### 第二步：用云中书生成排版 HTML

1. 打开 **[云中书 YunType](https://yuntype.pages.dev)**
2. 左侧粘贴 Markdown 内容，右侧选择配色 / 字体 / 比例
3. 点击 **导出 HTML**，复制生成的代码

> 云中书支持小红书风格多图排版、微信公众号长图文、信息图等多种格式。

---

### 第三步：在公众号编辑器里编辑 HTML

1. 用安装了插件的浏览器打开微信公众号后台，新建图文
2. 编辑器右侧出现 **`</> HTML` 边缘标记**，点开侧边面板（或按 `Alt+H`）
3. 点 **「⟳ 读取文章」** 载入当前正文 HTML，或直接粘贴你的排版代码
4. 编辑代码，下方实时预览效果
5. 点 **「应用到文章」** 写回正文；收起面板即见效果 🎉
6. 需要再调整时再打开面板，显示的就是当前最新代码，随时改、随时应用

### 关于图片 🖼

微信只认它自己 CDN 上的图片（外链图发布会被清掉），所以图片要先经过微信上传：

1. 用公众号顶部菜单的 **「图片」** 按钮，把要用的图先传进文章（堆在哪都行）
2. 打开本插件面板 → 点 **「🖼 图片库」→「⟳ 刷新」**，文章里的图就列成缩略图
3. 把光标放到代码里想要的位置，**点缩略图** —— 自动插入带真实微信链接的 `<img>` 标签
4. 应用后，图片就精确落在你 HTML 设计的位置上，发布也不会丢

## 🔧 技术原理

微信公众号编辑器基于 **ProseMirror**，内容承载在 `.ProseMirror` 这个 `contenteditable` 元素里。

**双向源码**：
- **读**：`editor.innerHTML` 直接取出文章当前的真实 HTML（含已上传图片的 `mmbiz` CDN 链接）
- **写**：`editor.innerHTML = 新代码` + 派发 `input` 事件，让 ProseMirror 接管；并清理其规范化时可能多出的空首行

**图片为什么可靠**：插件不去逆向微信的上传接口（易随版本失效），而是直接扫描 `editor.querySelectorAll('img')` 拿到文章里**已经合法**的 CDN 链接，再交给你在 HTML 里引用。上传走微信原生流程，链接天然有效、发布不被清除。

**安全**：应用前用正则剥离 `<script>` / `<iframe>` / `<object>` / `onXxx` 事件处理器；预览用 `sandbox` 隔离的 iframe 渲染。

## ⚠️ 注意事项

- 本插件以**开发者模式**加载，属于本地扩展，不会上传到 Chrome 商店，数据完全本地处理
- 微信公众号保存发布后，部分复杂 CSS 可能被服务器端二次过滤，建议使用云中书的**内联样式**导出版本
- 仅支持 `https://mp.weixin.qq.com` 域名下的编辑器页面
